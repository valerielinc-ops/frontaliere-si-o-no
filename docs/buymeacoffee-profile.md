## Bio / About

```
🇨🇭🇮🇹 Il primo simulatore fiscale gratuito per frontalieri Svizzera-Italia.

Confronta stipendio netto, tasse, pensione, costo della vita e molto altro tra Permesso G e Permesso B — tutto in un'unica app, 100% gratuita e senza pubblicità invasiva.

Ogni caffè ci aiuta a:
☕ Mantenere i server e le API attivi
📊 Aggiornare i dati fiscali ogni anno
🆕 Sviluppare nuove funzionalità
🚫 Tenere il sito senza pubblicità

Siamo un progetto indipendente nato per aiutare chi ogni giorno attraversa il confine. Se il nostro simulatore ti ha fatto risparmiare tempo (o soldi!), offrici un caffè ☕
```
